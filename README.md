# chopstick

* It is demo of a to do list tool.
* It uses react-js+localstorage+wepack
* It can： 
  * create a todo task/project
    1. add a name
    2. press enter to create a tag
    3. choose time
    4. add notes
  * delete or restore a task/project with file structure record
